# 个人博客系统规格说明

## 1. 项目概述

**项目名称**: Beeblog  
**项目类型**: 现代化个人博客系统  
**核心功能**: 支持 Markdown 渲染的文章发布系统，包含前台展示与后台管理  
**目标用户**: 个人博主、技术写作者

## 2. 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **Markdown**: react-markdown + remark-gfm
- **编辑器**: @uiw/react-md-editor (Markdown 编辑器)
- **数据存储**: 本地文件系统 (content/posts/*.md)

## 3. 功能列表

### 前台页面
- [ ] **首页/文章列表页** (`/`)
  - 展示所有文章卡片
  - 包含标题、摘要、发布日期、标签
  - 支持按标签筛选

- [ ] **文章详情页** (`/post/[slug]`)
  - 完整 Markdown 渲染
  - 代码高亮
  - 文章元信息展示

- [ ] **标签分类页** (`/tags`)
  - 所有标签列表
  - 标签下文章数量

- [ ] **搜索页面** (`/search`)
  - 全文搜索（标题、摘要、内容）
  - 搜索结果高亮匹配位置

- [ ] **关于页面** (`/about`)
  - 个人介绍
  - 联系方式

### 后台管理
- [x] **登录页** (`/admin/login`)
  - 简单密码验证

- [x] **管理后台** (`/admin`)
  - 文章列表管理
  - 新建/编辑/删除文章
  - Markdown 编辑器
  - **关于页面编辑**
    - 名称、标题、副标题
    - 头像 Emoji
    - 联系方式（邮箱、GitHub、Twitter）
    - 正文内容（Markdown）
  - **图片上传功能**
    - 支持点击上传本地图片
    - 支持粘贴剪贴板截图
    - 支持拖拽图片到编辑器
  - **API**: `/api/upload` 图片上传接口, `/api/about` 关于内容接口

## 4. 设计风格

- **主题**: 深色极简风格 (Dark Mode First)
- **配色方案**:
  - 背景: #0f172a (深蓝黑)
  - 卡片: #1e293b
  - 主色调: #38bdf8 (天蓝色)
  - 文字: #f1f5f9 (浅灰白)
  - 强调: #f472b6 (粉色)
- **字体**: Inter + JetBrains Mono (代码)
- **动效**: 页面渐入、悬停提升效果

## 5. 数据格式

Markdown 文件头格式 (frontmatter):
```yaml
---
title: 文章标题
date: 2024-01-01
tags: [前端, React]
excerpt: 文章摘要
---
```

## 6. 验收标准

1. 文章列表正确显示所有文章
2. 文章详情页正确渲染 Markdown
3. 标签筛选功能正常
4. 后台可登录并管理文章
5. 新建/编辑/删除文章功能完整
6. 响应式设计，移动端适配
